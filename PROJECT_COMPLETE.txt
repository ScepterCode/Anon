                    QUICK START (5 MINUTES)
================================================================================

1. SETUP ENVIRONMENT:
   python -m venv venv
   venv\Scripts\activate

2. INSTALL DEPENDENCIES:
   pip install -r requirements.txt

3. CONFIGURE ENVIRONMENT:
   copy .env.example .env
   # Edit .env with your Supabase credentials

4. CREATE SUPABASE TABLE (in Supabase SQL Editor):
   CREATE TABLE reports (
     id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
     description TEXT NOT NULL,
     image_url TEXT,
     category TEXT,
     location TEXT,
     status TEXT DEFAULT 'new',
     created_at TIMESTAMP DEFAULT now()
   );

5. CREATE STORAGE BUCKET:
   Supabase â†’ Storage â†’ New bucket â†’ Name: "report_uploads" â†’ Make public

6. CREATE ADMIN USER:
   python manage.py create_superuser --username admin --password admin123

7. RUN SERVER:
   python manage.py runserver

8. ACCESS:
   Form:  http://localhost:8000/reports/submit/
   Admin: http://localhost:8000/admin/login/
